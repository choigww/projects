## Problem

- 코로나19로 가속화된 온라인 생활, 그 중심에 있는 인터넷 커뮤니티
- 각각의 인터넷 커뮤니티는 매우 다양한 관심사를 다루는 서브채널(게시판)이 존재
- 단순 관심사를 포함해서, 내 성격이나 생활 패턴 등 다양한 요소를 고려해주는 커뮤니티 추천이 필요해!



## Solution

- 문자열 기반 인터넷 커뮤니티 게시판 추천 시스템
- 크게 두 가지 프로세스로 진행
  - 감성분석 기반 커뮤니티 추천
    - 문자열 감성분석 점수 기반
    - 일 평균 게시물 수/ 부정어 빈도 비율 등 다양한 요소 반영
  - 토픽모델링 기반 게시판 추천
    - 사용자의 주요 관심사 카테고리 기반
